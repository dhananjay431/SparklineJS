Line Charts

Smooth Line (default)

data: [1, 5, 2, 8, 3, 7, 4]
Sparkline.line(data, { tooltip: true })

Straight Line

smooth: false, color: #8b5cf6
Sparkline.line(data, { smooth: false })

Line + All Dots

showDots: true, color: #f59e0b
Sparkline.line(data, { showDots: true })

Line + Normal Range

normalRange: [3, 7]
Sparkline.line(data, { normalRange: [3,7] })

Line + Zero Line

negative data, zeroLine: true
Sparkline.line(data, { zeroLine: true })

Custom Size + Color

width: 280, height: 80, color: #06b6d4
Sparkline.line(data, { width: 280, height: 80 })

Area Charts

Area (blue)

color: #3b82f6, fillOpacity: 0.2
Sparkline.area(data, { color: "#3b82f6" })

Area (green)

color: #10b981, fillOpacity: 0.25
Sparkline.area(data, { fillOpacity: 0.25 })

Area + Tooltip

tooltip: true, color: #ec4899
Sparkline.area(data, { tooltip: true })

Area Straight

smooth: false, color: #f97316
Sparkline.area(data, { smooth: false })

Bar Charts

Bar (green)

color: #10b981, tooltip: true
Sparkline.bar(data, { tooltip: true })

Bar (red)

color: #ef4444
Sparkline.bar(data, { color: "#ef4444" })

Tristate

positive/negative/zero bars
Sparkline.tristate(data)

Tristate + Tooltip

tooltip: true
Sparkline.tristate(data, { tooltip: true })

Pie & Bullet Charts

Pie

data: [30, 20, 15, 25, 10]
Sparkline.pie(data, { tooltip: true })

Donut

donut: true
Sparkline.pie(data, { donut: true })

Pie Custom Colors

custom color palette
Sparkline.pie(data, { colors: [...] })

Bullet — Sales vs Target

value: 65, target: 80, max: 100
Sparkline.bullet(65, 80, 100)

3 Series Overlay

3 overlaid line series
Sparkline.multiLine(series, null, opts)

2 Series — Custom Colors

per-series color overrides
Sparkline.multiLine(data, seriesOpts)
